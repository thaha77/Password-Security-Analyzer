# ShadowAI Pass-Analyzer

This is a simple web-based project that checks how strong your password is. I built this to help people see if their passwords are safe from hackers.

## Tech Used
- HTML for the structure
- CSS for the dark hacker styling
- Vanilla JavaScript for checking the strength logic

## Features
- Real-time password strength checking.
- Color-changing indicator bar (Red for weak, Green for unbreakble).
- Calculates the estimated time a hacker needs to crack the password.
- Show/Hide password toggle button.

## How to Test
1. Clone this repository or download the files.
2. Open `index.html` in any web browser.
3. Type a password and check the status!
